# dashgo
dashgo is a admin painel, a project from module 4 of Ignite course, Rocketseat.

## 🔨 Features
- Authentication
- Dashboard
- List users
- Register users

## ✔️ Getting Started
`git clone https://github.com/ViniciusHack/dashgo.git`
`cd dashgo`
`yarn`
`yarn dev`
to start the delopment mode

## 🔧 Technologies
- Next.js
- ChakraUI
- React hook form
- MirageJS
- React Query
- JWT
